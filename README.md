# Numerical Methods in Python

This repository contains implementations of numerical methods developed as part of university coursework in Applied Physics.

## Contents

### Root finding methods (nonlinear equations)
- Bisection Method
- Newton Method
- Secant Method
- Fixed-point Iteration

### Systems of equations
- Fixed-point Iteration for Systems of Linear Equations (SLE)
- Fixed-point Iteration for Systems of Nonlinear Equations (SNLE)

### Other numerical methods
- Interpolation (Lagrange, Newton)
- Numerical Integration (Rectangle, Trapezoid, Simpson)
- Euler and Runge–Kutta Methods for Ordinary Differential Equations

## Tools used
- Python
- SymPy
- Matplotlib

## Purpose

Educational implementation of numerical algorithms to understand their behavior and practical usage.
